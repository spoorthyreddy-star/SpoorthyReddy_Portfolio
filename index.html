Publications

Electricity Theft Detection in Power Grids using Deep Learning and Random Forests

IJMMSA, Vol.14, Issue 3, July 2023 · Jul 29, 2023

Show Publication

Co-authored a research paper addressing non-technical losses (NTLs) in smart grids by detecting electricity theft using hybrid CNN–Random Forest (CNN-RF) models. Designed a Convolutional Neural Network to automatically extract temporal consumption features from smart meter data, reducing reliance on manual feature engineering. Integrated a Random Forest classifier optimized via grid search to improve theft detection accuracy, outperforming conventional ML and DL models. Applied SMOTE for data balancing and evaluated performance using confusion matrix, ROC, precision, recall, F-score, and accuracy. Achieved ~100% detection accuracy on real-world datasets (SEAI & LCL), demonstrating the model’s efficiency in reducing financial losses and enhancing power grid security.
