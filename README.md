## Overview

This portfolio presents a comprehensive profile of a Full-Stack Developer, AI/ML enthusiast, and Data Scientist. It highlights:

* Professional skills and expertise
* Featured projects with GitHub links
* Education and work experience
* Publications and certifications
* Contact and social links

It is designed to be **user-friendly**, **visually appealing**, and **responsive** across devices.

---

## Features

* **Hero Section:** Introduction with name, role, GPA, certifications, projects count.
* **About Me:** Personal and professional summary.
* **Services:** Overview of professional offerings, including Full-Stack Development, AI & Machine Learning, and Data Engineering & Visualization.
* **Projects:** Showcases key projects with descriptions, technology tags, and GitHub links.
* **Skills:** Technical skills and tools with icons (React, Python, TensorFlow, Node.js, etc.).
* **Education & Experience:** Timeline of degrees and work roles.
* **Publications:** Research papers with links to full PDFs.
* **Certifications:** Verified Coursera certifications with links.
* **Contact:** Email, LinkedIn, and embedded Google Maps location.
* **Animations:** Scroll reveal effects, hover animations on service cards, and a typewriter effect in the hero section.

---

## Technologies Used

* **Frontend:** HTML5, CSS3, JavaScript
* **Icons:** Lucide, Font Awesome, Devicon
* **Animation:** Intersection Observer API
* **Frameworks/Libraries:** None (pure vanilla JS)
* **Deployment Ready:** Can be hosted on GitHub Pages or any static hosting service

## Installation

To run locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/spoorthyreddy-star/portfolio.git
   ```
2. Navigate to the project directory:

   ```bash
   cd portfolio
   ```
3. Open `index.html` in a browser.
